# 🪨 Marble Store – E-Commerce Website

An **E-Commerce Website for Marble Products** where users can explore a variety of marbles, view product details, and place orders easily. The website allows customers to browse marble collections and place orders through a simple and user-friendly interface.

This project demonstrates the use of **frontend technologies along with PHP for backend functionality**.

---

## 🚀 Features

* Browse different types of marbles
* View product images and details
* Place orders through the website
* Dynamic functionality using PHP
* Clean and user-friendly interface
* Lightweight and easy to run

---

## 🛠️ Technologies Used

* **HTML** – Structure of the website
* **CSS** – Styling and layout
* **JavaScript** – Client-side interactions
* **PHP** – Backend processing and order handling

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/yourusername/marble-store.git
```

2. Navigate to the project folder

```bash
cd marble-store
```

3. Run the project using a **local server (such as XAMPP or WAMP)** since the project uses PHP.

4. Open the project in your browser.

Example:

```
http://localhost/marble-store
```

---

## 🛍️ How It Works

1. Users visit the website.
2. They can **browse available marble products**.
3. Select the marble they want to purchase.
4. Place an order using the website's order system powered by **PHP**.
